# CP1.2 – Jenkins CI

Este repositorio contiene un proyecto sencillo en Python que se utiliza como base para el **Caso Práctico 1.2** del módulo de DevOps.

El objetivo del proyecto es practicar distintos tipos de pruebas software, como pruebas unitarias, pruebas de integración y pruebas de rendimiento, así como su integración dentro de un pipeline de **Integración Continua con Jenkins**.

El código y la estructura del proyecto son simples para facilitar la comprensión de los conceptos trabajados durante el curso y poder centrarse en el uso de las herramientas de CI y testing.

Este repositorio se utiliza como fuente de código para la ejecución de un pipeline de Jenkins, donde se automatiza la ejecución de las pruebas y se analizan los resultados obtenidos.

## Tecnologías utilizadas

* Python
* Pytest
* Flask
* Jenkins
* JMeter
